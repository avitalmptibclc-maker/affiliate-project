# 🌺 יועצות ההנקה של אילת - Lactation Hub Eilat

## 🛠️ התקנה ושימוש

### 1. התקנת תלויות
```bash
npm install
```

### 1.1. קונפיגורציית Supabase
צרי קובץ בשם `.env.local` בתיקיית השורש עם הערכים האלה:

```env
VITE_SUPABASE_URL=https://ucodslcraqotfkjpbqvm.supabase.co
VITE_SUPABASE_ANON_KEY=YOUR_SUPABASE_ANON_KEY
```

הקובץ `.env.local` נוסף ל־`.gitignore`, כך שהוא לא יעלה לגיט.

### יצירת הטבלאות ב‑Supabase
פתחו את `SQL Editor` ב‑Supabase והפעילו את הפקודות הבאות:

```sql
create table consultants (
  id bigint generated by default as identity primary key,
  name text not null,
  credentials text,
  statement text,
  location text,
  channels jsonb,
  additionalFields text,
  phone text,
  email text,
  image text,
  created_at timestamptz default now()
);

create table pending_consultants (
  id bigint generated by default as identity primary key,
  name text not null,
  credentials text,
  statement text,
  location text,
  channels jsonb,
  additionalFields text,
  phone text,
  email text,
  image text,
  created_at timestamptz default now()
);
```

### 2. הרצת שרת הפיתוח
```bash
npm run dev
```
האתר יפתח באופן אוטומטי ב-http://localhost:3000

### 3. בניית לפרודקשן
```bash
npm run build
```

## 📖 איך משתמשים?

### דף הבית (Home)
- מציג את כל יועצות ההנקה המאושרות
- כל יועצת מהווה כרטיסיה אחד עם שמה, הסמכותיה, תמונה וכן השירותים שלה
- לחצנים ישירים ל-WhatsApp וקריאה טלפונית

### טופס הרשמה (Register)
- יועצות חדשות יכולות להרשם דרך טופס פשוט
- הם מוזנים למאגר זמני וממתינים לאישור

### פאנל ניהול (Admin)
- אביטל יכולה לראות את כל הבקשות שממתינות לאישור
- אפשרות להעלות תמונה לכל יועצת
- כפתור אישור שמוסיף את היועצת לדף הציבורי

## 🎯 המשך (לעתיד)

כדי לעשות את האתר במלואו פעיל בעולם האמיתי:

1. **Database** - החלף את ה-useState בקריאות API לנתונים אמיתיים (Firebase, MongoDB וכו')
2. **אימות** - הוסף קיד אימות לפאנל הניהול של אביטל (סיסמה / Google Auth)
3. **העלאת תמונות** - חברת עם שרת לטיפול בתמונות בפועל (S3, Cloudinary וכו')
4. **דיפלוי** - העלה לשרת (Vercel, Netlify, GitHub Pages וכו')
