# 🌺 דפי הנחיתה של אביטל — Landing Pages

ריפו אחד שמכיל **כמה דפי נחיתה במקביל**, כשכל דף עצמאי לחלוטין ואינו משפיע על האחרים.

---

## 🧭 איך הכל בנוי

```
landing-pages/
├── README.md                  ← הקובץ הזה (ספר ההוראות)
├── sites/                     ← כל דף נחיתה חי כאן, בתיקייה משלו
│   └── yoatsot-eilat/         ← דף "יועצות ההנקה של אילת"
│       ├── index.html         ← הדף עצמו (קובץ אחד שלם, עובד בכל מקום)
│       └── logo.png
├── _template-new-site/        ← תבנית להתחלת דף חדש (מעתיקים אותה)
│   ├── index.html
│   ├── logo.png
│   └── tables.sql             ← הטבלאות של הדף ב-Supabase
└── supabase/                  ← תיעוד הטבלאות של כל דף
    └── yoatsot-eilat.sql
```

### שלושת עקרונות הבידוד 🔒
כל דף מבודד מהאחרים בשלוש שכבות:

| שכבה | איך מבודד |
|------|-----------|
| **קוד** | כל דף = תיקייה נפרדת עם קובץ HTML אחד שלם. אין קוד משותף, אין ייבוא. |
| **פרסום (Vercel)** | כל דף = פרויקט Vercel נפרד שמצביע על התיקייה שלו בלבד. |
| **נתונים (Supabase)** | כל דף = טבלאות נפרדות עם קידומת שם משלו (למשל `telaviv_consultants`). |

פרויקט Supabase אחד ומפתח אחד משרתים את כל הדפים — אבל כל דף ניגש רק לטבלאות שלו.

---

## ➕ איך מוסיפים דף נחיתה חדש (4 צעדים)

### 1. צרי את תיקיית הדף
העתיקי את התיקייה `_template-new-site` לתוך `sites/` ותני לה שם באנגלית:
```
sites/
└── telaviv/          ← דוגמה לדף חדש
    ├── index.html
    ├── logo.png
    └── tables.sql
```

### 2. הגדירי את הדף
פתחי את `index.html` של הדף החדש, ובראש הקובץ שני שורה אחת בלבד:
```js
const SITE_PREFIX = "telaviv";   // 👈 קידומת ייחודית באנגלית, בלי רווחים
```
זה כל מה שצריך כדי שהדף ידבר עם הטבלאות שלו ולא של אף אחד אחר.
(בנוסף כדאי לעדכן את הכותרת `<title>` והטקסטים בדף לפי הנושא החדש.)

### 3. צרי את הטבלאות ב-Supabase
פתחי את `tables.sql` שבתיקייה, החליפי בכל הקובץ את `__PREFIX__` בקידומת שבחרת
(בעורך טקסט: "החלף הכל"), העתיקי הכל ל-**SQL Editor** ב-Supabase ולחצי **Run**.

### 4. פרסמי ב-Vercel
ב-Vercel צרי **New Project** מהריפו הזה, ובהגדרה **Root Directory** בחרי את התיקייה
של הדף (למשל `sites/telaviv`). כך כל דף מתפרסם בנפרד ובכתובת משלו.

---

## ✏️ עדכון דף קיים
ערכי את הקובץ `sites/<שם-הדף>/index.html`, שמרי, ועשי `git push`.
אם הדף מחובר ל-Vercel דרך הריפו — הוא יתעדכן אוטומטית.
