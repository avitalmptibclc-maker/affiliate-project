# 💼 תבנית "קישור בביו" — להקמה ללקוחות (link-in-bio)

תבנית מוכנה למכירה: דף "קישור בביו" יוקרתי עם 4 מגירות, מוצרים, ופאנל ניהול
שבו הלקוח/ה מנהל/ת הכל לבד (תמונה, רקע, שמות מגירות, הסברים, אייקונים, מוצרים)
— בלי לגעת בקוד.

הקרדיט בתחתית ("נבנה באהבה על ידי אביטל עציון") נשאר ומקשר לאתר שלך — מיתוג חינם בכל דף שתמכרי. 💜

---

## 🛠️ הקמה ללקוח/ה חדש/ה — 5 צעדים

### 1. עותק של התיקייה
העתיקי את `_template-linkbio/` לתוך `sites/` ותני שם באנגלית, למשל:
```
sites/dana/
├── index.html
└── tables.sql
```

### 2. Supabase
מומלץ **פרויקט Supabase נפרד לכל לקוח/ה** (הנתונים שלהם נשארים אצלם):
- צרי/בקשי פרויקט ב-supabase.com
- ב-`tables.sql`: החליפי `__PREFIX__` בקידומת (למשל `dana`) → הריצי ב-SQL Editor
- מתוך הפרויקט: **Project Settings → Data API / API Keys** — העתיקי את ה-**Project URL** ואת ה-**anon key**

### 3. הגדרות בקובץ `index.html`
בראש הקובץ, בבלוק ההגדרות, מלאי:
```js
const SITE_PREFIX = "dana";              // אותו ערך כמו ב-SQL
const ADMIN_CODE  = "1234";              // סיסמת הניהול של הלקוח/ה
const BRAND_NAME  = "דנה כהן";           // השם בכותרת
const HERO_TITLE  = "הי יקרות!";          // כותרת ראשית
const HERO_INTRO  = "משפט פתיחה קצר...";
const SUPABASE_URL = "https://xxxx.supabase.co";   // מהשלב הקודם
const SUPABASE_ANON_KEY = "eyJ...";                // מהשלב הקודם
```

### 4. פרסום ב-Vercel
- **Add New → Project** מהריפו → **Root Directory** = `sites/dana`
- (או, אם פורסים מהשורש, ה-`vercel.json` כבר מפנה לדף — אבל לכל לקוח עדיף פרויקט Vercel נפרד עם Root Directory משלו)
- מקבלים כתובת (אפשר גם לחבר דומיין אישי של הלקוח/ה)

### 5. מסירה ללקוח/ה
שלחי לה את הכתובת + את **קוד הניהול** (`ADMIN_CODE`). מכאן היא:
- 🔒 נכנסת לניהול עם הקוד
- מעלה תמונת פרופיל ורקע
- משנה שמות מגירות + כותבת הסברים + בוחרת אייקונים
- מוסיפה מוצרים (התמונה נמשכת אוטומטית מהקישור)

---

## ✅ מה כלול בכל דף
- עיצוב יוקרתי, מותאם מובייל, פונט עברי
- 4 מגירות נפתחות + הסבר לכל מגירה
- כרטיסי מוצר עם תמונה אוטומטית מהקישור
- העלאת תמונת פרופיל + רקע דרך הניהול
- בורר אייקונים (16 אייקונים)
- גילוי נאות לאפיליאייט + קרדיט מקושר אלייך

## 💡 טיפ למכירה
כל לקוח/ה = תיקייה + פרויקט Supabase + פרויקט Vercel נפרדים → בידוד מלא, אף אחת לא רואה את הנתונים של השנייה.
